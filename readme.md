# Errors
## 1x Success Codes
- `10` - Success

## 2x Authentication Error Codes
- `20` - Not Logged In
- `21` - Not Enough Permissions

## 3x Team Error Codes
- `30` - Team Not Found
- `31` - Team Full
- `32` - Not Team Leader
- `33` - User Already In Team
- `34` - User Didn't Request To Join

## 4x Endpoint Error Codes
- `40` - Not Found/Page Doesn't Exist
- `41` - Argument Error

## 5x User Error Codes
- `50` - User Not Found
- `51` - User Not Authorized
